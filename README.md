## Toggle

---

- 구현 방법

  - <button>에 onClick event에 함수를 주어 구현
  - classList.toggle을 주어 버튼과 배경이 움직이도록 구현함
  - state로 toggleBox 안에 text on/ off가 변하도록 함

- 실행 방법
  - 원형 버튼을 클릭 시 버튼이 좌 우로 움직이며 background 이미지 전환할 수 있음
